Проект по Практике Цифрового Производства в МФТИ. Устройство проявляет фотоплёнку. 

Комплектующие:
Arduino Uno
Нагреватель
Реле х2
Мотор JGB37-3525 12V
Насос 12V
Каркас из металлического профиля и стального квадрата
Джойстик
Дисплей st7735
Датчик температуры 
Блок питания 12V
Преобразователь напряжения 12-5V

За что отвечают пины ардуино:
2 - направление вращения мотора
5 - скорость вращения
4 - динамик
6 - реле на нагреватель
7 - реле на двигатель
8, 9, 10, 11, 13 - дисплей
A0, A1 - джойстик
A2 - датчик температуры
